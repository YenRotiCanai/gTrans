# gTrans
eng->zh-tw dictionary

## Environment
python3.6  
pip3  
googletrans  
pycharm

## Installation
1. Install python3
2. Install pip3
3. Install googletrans https://pypi.org/project/googletrans/
4. run gTrans.py

## Instruction
1. Type 1 to open the dict or 0 to exit and bye.
2. Type the english word and press Enter.
3. 繁中就會出來啦。
4. 如果要退出就輸入0。
